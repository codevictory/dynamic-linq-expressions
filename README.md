# Dynamic LINQ Expressions

Example code for blog post: https://codevictory.net/posts/linq/dynamic-linq-expressions/

## Installation

Needed dependencies:  
**SDK:** .NET Core 3.1  
**Other libraries:** System.Linq.Expressions (can be found in nuget)

Commands to run:
```
$ dotnet build
$ cd Application/
$ dotnet run
```

Have fun!
